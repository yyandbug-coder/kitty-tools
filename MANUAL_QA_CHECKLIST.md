# Kitty Tools 手动回归清单（Windows / macOS）

在发布或合并大改前，请在 **Windows** 与 **macOS** 各跑一遍。每项记录：**通过** / **失败** / **备注**。

自动化闸门（本地）：`pnpm check`、`pnpm check:rust`，或分别运行 `cargo clippy --all-targets -- -D warnings`、`cargo test`。  
GitHub：`/.github/workflows/ci.yml` 在 `macos-latest` 与 `windows-latest` 上跑 Rust；`ubuntu-latest` 上跑前端构建。  
（本地若未安装 Windows 目标，请在 Windows 机器上补跑 `cargo check`。）
